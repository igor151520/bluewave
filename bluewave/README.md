# 🌊 BlueWave

## 📌 Sobre o projeto
O BlueWave é uma aplicação web desenvolvida com React que tem como objetivo demonstrar um protótipo funcional de interface moderna, focada em organização de componentes e interatividade básica.

Este projeto foi desenvolvido como atividade avaliativa da disciplina de Front-end II.

---

## 🚀 Funcionalidades
- Exibição de elementos em tela (cards)
- Componentização da interface (Header, Card)
- Interação com o usuário através de eventos (botões)
- Atualização de estado utilizando React

---

## 🛠️ Tecnologias utilizadas
- React
- Vite
- JavaScript
- CSS
